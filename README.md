# higgs

Higgs monorepo — multi-app Turborepo for AI agent tooling and web apps.

## Stack

Docker, Turborepo, TypeScript

## Getting started

```bash
git clone https://github.com/ANSHSINGH050404/higgs.git
cd higgs
bun install
bun run dev   # or: bun run start
```
